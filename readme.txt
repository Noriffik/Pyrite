Pyrite - main folder. Contains Pyrite core, UI, "standart actions", HierarchicalObject projects in one solution;
PyriteRemoteForAndriod - contains application for andriod.
PyriteMods - contains custom "actions" and "checkers". Now "ModbusAction", "ModbusChecker", "VolumeAction";
ModsExample - examples of ICustomChecker and ICustomAction implementation;
Release - contains builds from "PyriteMods", "PyriteRemoteForAndriod" and "Pyrite".
